# go-rest-api-example

> Simple RESTful API built using Golang
> Uses mux package (github.com/gorilla/mux) to route requests

##  Installation
```
go get github.com/gurveerdhindsa/go-rest-api-example.git
```

## API
/people
- GET : Get all people

/people/{id}
- GET: Get a person with id
- POST: Add a person
- DELETE: Delete a person
